# 📍 Faculty Tracking System (WiFi-Based)

## 🚀 Overview

The **Faculty Tracking System** is a full-stack web application designed to track and locate faculty members within a college campus using WiFi network data. The system leverages network logs, real-time processing, and a user-friendly interface to help students and staff quickly find faculty locations.

---

## 🧠 Key Features

* 📡 **WiFi-Based Location Tracking**
  Uses access point data and signal strength to estimate faculty location.

* 🔍 **Search by Faculty Name**
  Quickly find the current location of any registered faculty member.

* ⚡ **Real-Time Updates**
  Continuously updates location data from the server.

* 🔐 **Secure Data Handling**
  Ensures privacy with encrypted data handling and restricted access.

* 💻 **Full-Stack Application**
  Built with a modern frontend and backend architecture.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/faculty-tracker.git
cd faculty-tracker
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
node server.js
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```
